Configuring UP. The lens consists of the following procedures:

<ol>
  <li>Configure Live Sources if they have not been set up yet. Live Sources are configured outside of the UP. Lens application, via Live Sources Services. See \[cross-ref will go here].\[Reviewers: We will have a UP configuration-specific chapter and/or online section that will describe how to configure Live Sources. We’ll add a cross-reference to that when it’s ready.]</li>
  <li>Configure a monitoring template. In a monitoring template, you select the monitoring categories and the conditions to monitor. See Monitoring Templates.</li>
  <li>Create and activate sensors, which are the fundamental monitoring components of UP. Lens. Sensors act as “eyes on the ground” that ingest live video feeds to provide real-time observability and deep technical analysis.</li>
  <li>(Optional) Add sensor tiles and widgets to a multiviewer dashboard. See Manage Multiviewers.</li>
</ol>
